# Incident Response Playbook — NIST SP 800‑61 aligned

**Résumé:** Plan d'intervention structuré basé sur le guide NIST 800‑61 pour détecter, analyser, contenir, éradiquer et restaurer les systèmes après un incident.

## Contenu du dépôt
- `playbook.md` — plan complet IRP.
- `templates/` — modèles pratiques à copier lors d’un incident.
- `runbooks/` — procédures spécifiques (ransomware, C2, etc.).
- `samples/` — exemples de timeline et rapports.
